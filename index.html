Creating a Repository

To create new repository, follow this steps

  1. Go to your GitHub account
  2. Click on New Repository.
  3. Name your repository, add a description, and choose its visibility (public/private)

Some Git command

git init: This initializes an empty Git repository in your current directory.

git clone: Clone a Repository

git checkout -b: create and switch into a branch

git add : Stages changes et.c
